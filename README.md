# first-project
This will be my first project that I attempt on GitHub. I aim to make a study planner service that will allow students such as myself to be able to use this during the school year.

dev-log file: This will contain a running entry of
all the changes I made on a day-by-day basis.

ClassRoom file: This file contains the ClassRoom class,
which on the home page will represent a unique class in
my schedule. This class contains a name, and a list
of assignments. All of the instance variables contain
getter and setter variables, or whatever would be
equivalently appropriate for each.

Assignment file: This file contains the Assignment class. Each assignment will be contained under a list in a ClassRoom object.
Each will detail the name, due date, and completedness of the 
assginment. All instance variables have appropriately named getters and setters.
